# Chapter 64

## Text

其安易持，其未兆易謀。其脆易泮，其微易散。爲之於未有，治之於未亂。合抱之木生於毫末。九層之臺起於累土。千里之行始於足下。爲者敗之，執者失之。是以聖人無爲故無敗，無執故無失。民之從事常於幾成而敗之。慎終如始則無敗事。是以聖人慾不欲，不貴難得之貨。學不學，復衆人之所過，以輔萬物之自然而不敢爲。
