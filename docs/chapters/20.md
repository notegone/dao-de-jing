# Chapter 20

## Text

絕學無憂，唯之與阿，相去幾何？善之與惡，相去若何？人之所畏，不可不畏。荒兮其未央哉！衆人熙熙如享太牢、如春登臺。我獨泊兮其未兆，如嬰兒之未孩；儡儡(lěi,羸弱)兮若無所歸。衆人皆有餘，而我獨若遺。我愚人之心也哉！沌沌兮。俗人昭昭，我獨昏昏；俗人察察，我獨悶悶。衆人皆有以，而我獨頑且鄙。我獨異於人，而貴食母。
