# Chapter 38

## Text

上德不德是以有德。下德不失德是以無德。上德無爲而無以爲。下德無爲而有以爲。上仁爲之而無以爲。上義爲之而有以爲。上禮爲之而莫之以應，則攘臂而扔之。故失道而後德。失德而後仁。失仁而後義。失義而後禮。夫禮者忠信之薄而亂之首。前識者，道之華而愚之始。是以大丈夫，處其厚不居其薄。處其實，不居其華。故去彼取此。
