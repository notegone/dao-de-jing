# Chapter 31

## Text

夫佳兵者不祥之器，物或惡之，故有道者不處。君子居則貴左，用兵則貴右。兵者不祥之器，非君子之器，不得已而用之，恬淡爲上。勝而不美，而美之者，是樂殺人。夫樂殺人者，則不可得志於天下矣。吉事尚左，凶事尚右。偏將軍居左，上將軍居右。言以喪禮處之。殺人之衆，以悲哀泣之，戰勝以喪禮處之。
